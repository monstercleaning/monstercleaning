## Public projects

### [gsc](https://github.com/monstercleaning/gsc) — A Pre-Registration Reproducibility Stack for Falsifiable Cosmological Models

An open-source experiment in multi-LLM iterative hostile-review as a research workflow, demonstrated on a scale-covariant cosmology framework. After two AI hostile-audit cycles, most predictions fail current observational data — which is itself the methodological point: pre-registration discipline catches errors before submission, retracts them transparently, and updates framework status without spin.

- 10 pre-registered numerical predictions (P1–P10) with deterministic compute pipelines
- Schema-validated JSON artifacts, lineage DAGs, content hashing
- Cryptographically-signed prediction protocol with append-only register
- Three companion paper drafts (empirical, theoretical, methodology)
- MIT licence
